FRONT-END Technologies

New version of HTML with larger set of technologies that allows more diverse and used to build powerful website and applications.

CSS3 is the latest evolution of the Cascading Style Sheets language and aims at extending CSS2.1. It brings a lot of long-awaited novelties, like rounded corners, shadows, gradients, transitions or animations and so on.

Most well-known as the scripting language for Web pages, many non-browser environments also use it, such as node.js and Apache CouchDB.

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

AngularJS is a JavaScript framework. It can be added to an HTML page with a script tag. AngularJS extends HTML attributes with Directives, and binds data to HTML with Expressions.

SQL is a standard language for accessing databases.Used to manipulate data in: MySQL, SQL Server, Access, Oracle, Sybase, DB2, and other database systems.

Node.js is a very powerful JavaScript-based framework/platform built on Google Chrome's JavaScript V8 Engine.

Bootstrap is the most popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first web sites.
